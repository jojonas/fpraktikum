Dieses LaTeX-Paket stellt eine fuer studentische Angelegenheiten als praktisch erwiesene Sammlung
von Formatierungen fuer LaTeX2e zur Verfuegung. 

1. Dateien: Montavon.cls, legendde.tex, README.txt
2. LaTeX-Pakete, die installiert sein muessen:
	- fontenc
	- inputenc
	- lmodern
	- babel
	- fancyhdr
	- graphicx
	- amsmath
	- amssymb
	- upgreek
3. Neue Kurzbefehle
	- \heads{}{}{} = Kopfzeilen Links, Mitte, Rechts
	- \legendde = Notationslegende aus legendde.tex
	- \li = linke Klammer
	- \re = rechte Klammer
	- \pdiff{}{} = partiell differenziert
	- \diff{}{} = dfferenziert 
	- \di = Differential d
	- \poi{} = Poisson-Klammer
	- \ord{} = Ordnung
	- \reell = reelle Zahlen R
	- \svec{}{}{} = Spaltenvektor 3D
	- \tdvec{}{} = Spaltenvektor 2D
	- \Kern = Kern (Matrix)
	- \Norm{} = Betragsstriche
	- \Normm{} = Doppelte Betragsstriche
	- \vecb{} = fettgedruckter, kursiver Vektor
	- \nullvec = Nullvektor
	- \matb{} = fettgedruckte, aufrechte Matrix
	- \vecbe{} = Einheitsvektor e_?
	- \uuline{} = Doppelt unterstrichten
	- \im = imaginaere Einheit
	- \expe = Exponentialfunktion aufrechtes e
	- \unit{} = Aufrechte Einheit mit Abstand
	- \pow{} = Zehnerpotenz
	- \\ = Neue Zeile ohne Einrueckung
